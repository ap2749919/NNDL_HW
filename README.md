# NNDL_HW
24春季神经网络与深度学习 作业

训练：main.py中的train_model()函数，参数search为是否进行参数搜索，默认为0，即不进行搜索，直接取测试的最佳参数；如果设置为1则进行搜索，并将搜索结果输出到tuning_results.txt，同时将最佳参数自动套入后续训练；

测试：main.py中的test_model()函数，加载已经存储的npz参数文件进行预测。
